Getting the API Key

The first step to Yelp API is getting the API Key. But what is an API Key? API Key is normally given after you have created an account with the service provider. It is a unique value that must be included along with the GET or POST request (more about this in "Using API"). This key allows the service provider to identify who is sending the request so that they can keep track of how many requests you are sending.

Follow these instructions on how to get the API Key:

Step 1: Create a Yelp account. Sign up for an account here.

Step 2: Gain access to Yelp API Key here.
