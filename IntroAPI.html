What is an API?

Application Programming Interface

According to Microsoft, Web APO "is a framework that makes it easy to build HTTP services that reach a broad range of clients, including browsers and mobile devices." What exactly does that mean? Think of cloud computing. Cloud computing is a network of remote servers hosted on the internet to store, manage, and process data. Web API is similar to this. In Web API, anyone with some experience in JavaScript can 'expose' their data through Web API to other people via browsers and other devices such as mobile apps and tablets, for others to access this data. It is an open source for providing data to other people. It is like an online library, such as Wikipedia, where people can contribute data and others can access that data.

What does Yelp API do?

Now that you know a little about Web API, let's look at what Yelp API does. Yelp API allows you to pass search parameters such as "find term", "category", "near location", and other more search words. The response from that search includes useful business information such as business name, phone, address, and Yelp rating. It will include a list of businesses that best match the criteria of the search parameters. With the search results on hand, this will give you helpful information on other businesses that you can use for consumption or business-related purposes.

Here are some of the things you can do with Yelp API:

  • Search for businesses by keyword, location, or category.
  • Search for businesses by phone.
  • Search for businesses by services offered such as delivery or pickup.
  • Find business information such as photos, rating, price levels, and hours of operation.
